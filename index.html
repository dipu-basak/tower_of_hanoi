Problem Formulation Definition

Core Formulation

The problem models mathematical sequences of optimization. The target is to systematically isolate tasks into minimal substeps and compute exact move instructions cleanly.

It allows engineering minds to evaluate boundaries, parameters, data transformations, and state validation sequences efficiently.

Supplementary Myth

Invented in 1883 by Édouard Lucas. Legend notes priests moving 64 golden disks across three rods. When done, the world ends. Solving it requires 264 - 1 steps (585 billion years).

04 / 16

Problem & Constraints

Our objective is to move a stack of n disks from the Source Rod (A) to the Destination Rod (C) using an Auxiliary Rod (B).

1. Single Move

Only one disk can be moved at a time from one rod stack to another.

2. Top Pick

A move consists of picking up only the top disk and placing it onto another stack.

3. No Overlap

No larger disk can ever be placed on top of a smaller disk.

Understanding Recursion

What is Recursion?

Recursion is a foundational programming paradigm where a function continuously calls itself to solve smaller instances of the same computational problem.

Instead of linear loops, it builds nested operational tracking pipelines that unwind sequentially once computing thresholds are fully met.

The Key Components

• Base Case: The termination hook that stops structural looping instantly (e.g. n == 1).

• Recursive Call: The statement driving the program parameters closer towards the Base Case condition.

Our Stack-Based Code

def hanoi(n, a, b, c) -> list: if n == 1: # Base Case: Move disk from a to c c.append(a.pop()) else: # Step 1: Move top n-1 disks from a to b hanoi(n - 1, a, c, b) # Step 2: Move the largest disk from a to c c.append(a.pop()) # Step 3: Move the n-1 disks from b to c hanoi(n - 1, b, a, c) return c

Logic Flow Analysis

Base Case (n = 1): Simply moves a single disk from Source directly to Destination.

Recursive Step (n > 1): Solves the subproblem of moving n-1 disks out of the way, executes one base movement, and reconstructs the stack on target.
